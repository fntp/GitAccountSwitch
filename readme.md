# Git Account Switcher

A Visual Studio Code extension that allows you to easily switch between different Git accounts and configurations.

## Features

- Quickly switch between multiple Git user configurations.
- Supports global and local Git settings.
- Integrates seamlessly with VSCode's command palette.

## Installation

1. Open VSCode.
2. Go to Extensions (Ctrl+Shift+X).
3. Search for "Git Account Switcher".
4. Click Install.

## Usage

1. Open the Command Palette (Ctrl+Shift+P).
2. Type "Switch Git Account" and select the desired account.
3. The extension will update your Git configuration accordingly.
